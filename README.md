# 使用方法
1.開啟 git bash. 

2.更改目錄到您想要下載此專案的位置 
```
> cd (your directory)
```
3.下載此專案 
```
> git clone https://github.com/wei-0321/VTuber.git
```
4.Change the diretory to this repository.
```
> cd VTuber
```
5.Execute the program.
```
> python GUI.py
```
